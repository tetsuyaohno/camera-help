# 黒板編集 操作マニュアル

このマニュアルでは、BoadEditActivity における「黒板編集」画面での編集操作および各ボタン操作について解説します。

---

## 1. 編集操作

### 1.1 テキスト入力
- **対象フィールド**:
  - 工事名: 工事名を入力します。　工事登録で工事番号・工事名を登録している場合、工事番号、工事名の順に入力されています。
  - 工種: 工種を入力します。工事登録で工種を登録している場合は、タップすると候補リストが表示されます。
  - 測点: 測点を入力します。工事登録で測点を登録している場合は、タップすると候補リストが表示されます。
  - 内容: メイン部分の文字列を入力します。
  - すべてのフィールドの入力は任意であり入力がないと空白になります。
- **プレビュー反映**:
  - 入力中の文字はプレビュー黒板に即時反映されます。fileciteturn1file0 fileciteturn1file4

### 1.2 文字サイズ変更
- **プリセット切替**:
  - 「文字サイズ」欄で「小／中／大／特大」を選択すると、フォントサイズが 14f／22f／30f／40f に設定されます。fileciteturn1file0 fileciteturn1file4
- **直接入力**:
  - 「文字サイズ」欄に数値を入力すると、プレビュー中の「内容」のフォントサイズが更新されます。fileciteturn1file0

### 1.3 文字配置変更
- **横位置**:
  - 「左／中央／右」を選択すると、それぞれ「内容」欄の文字の水平配置が変更されます。fileciteturn1file0
- **縦位置**:
  - 「上／中／下」を選択すると、それぞれ「内容」欄の文字のの垂直配置が変更されます。fileciteturn1file0

### 1.4 入力補助（候補リスト）
- **工種・測点候補**:
  - 工種欄または測点欄をタップまたはフォーカスすると、下部に候補リストが表示されます。
  - リストから選択できるほか、通常の文字入力にも対応しています。fileciteturn1file0 fileciteturn1file2

## 2. ボタン操作

### 2.1 戻る／撮影戻るボタン
- **戻る↩️／撮影戻る🟩ボタン**: （どちらかが表示されています）
- **操作**: いずれも、前画面へ戻ります。fileciteturn1file0 fileciteturn1file4

### 2.2 新規黒板選択 ボタン
- **操作**: ベース黒板選択画面が表示されるので、「boad1　～　boad3」 のベース黒板を選択します。工事番号・工事名が登録されている場合には記入されています。fileciteturn1file0 fileciteturn1file1

### 2.3 マイ黒板読み込み ボタン
- **操作**: 保存済み黒板リストが表示されます。使用したい黒板を選択すると元の画面に戻り、選択した黒板が表示されています。必要に応じて内容を書き換えます。fileciteturn1file0 fileciteturn1file3

### 2.4 上書保存 ボタン
- **操作**: 黒板を編集した場合で新たにマイ黒板を生成しない場合はこのボタンで保存します。元の内容は失われます。完了すると「上書き保存しました」とトースト表示。※新規に作成した黒板の場合は新規保存されます。fileciteturn1file0

### 2.5 新規保存 ボタン
- **操作**: 新規に黒板を作成した場合、編集した黒板を新たに保存（変種前の黒板も残す）する場合、新規に保存されマイ黒板一覧に追加されます。Officeファイルなどの「名前を付けて保存」にほぼ同じ。fileciteturn1file0

### 2.6 削除 (buttonDel)ボタン
- **操作**: 確認ダイアログ表示後、「削除」を選択で黒板を削除し、画面を初期状態にリセット。キャンセルで操作中止。fileciteturn1file0
